# Smart Campus AI

A full-stack application featuring a React frontend, a Node.js/Express backend (with Prisma and SQLite), and a Python/FastAPI chatbot service.

## Project Structure
- `/frontend` - React application (Vite + Tailwind CSS)
- `/backend` - Node.js Express server + Prisma ORM (SQLite)
- `/chatbot` - Python FastAPI microservice

## Setup Instructions
Please refer to each directory for specific setup instructions.

### Prerequisites
- Node.js (v18+ recommended)
- Python (3.9+ recommended)
